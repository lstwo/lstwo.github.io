Projects

All projects lstwoSTUDIOS is currently working on / has worked on in the past:

The Wobblyrooms

The Wobblyrooms is a Wobbly Life mod aimed to create an accurate backrooms mod. It features many levels and you can check out the wiki for more information right here!

The mod can be downloaded on the Steam workshop for Wobbly Life, to be played in the arcade game mode here!

The Puzzle Rooms

The Puzzle Rooms is a Wobbly Life arcade mod containing many rooms with puzzles / challenges for you to complete!

The mod was made in about 4-5 days actually has lore.

The mod can be downloaded on the Steam workshop for Wobbly Life, to be played in the arcade game mode here!

Lightsaber Battle

The Lightsaber Battle mod is a mod in which you can compete against your friends in an arena and fight with lightsabers.

By killing players you can gain global kills which save between sessions. You can use these to unlock new lightsabers and maps!

The mod can be downloaded on the Steam workshop for Wobbly Life, to be played in the arcade game mode here!
